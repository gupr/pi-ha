Vanliga problem och hur du löser dem

Pi startar inte / ingen kontakt

Problem: Du kommer inte åt Home Assistant.

Lösning:

  • Vänta minst 10 minuter första gången
  • Kontrollera att strömadaptern är korrekt
  • Kontrollera att Ethernet-kabel är inkopplad
  • Testa annan webbläsare
  • Testa IP istället för namn (kolla i router)
Kan inte öppna homeassistant.local

Problem: Webbadressen fungerar inte.

Lösning:

  • Använd istället IP-adressen (t.ex. 192.168.x.x:8123)
  • Kolla i din routers enhetslista
  • Se till att dator och Pi är på samma nätverk
Zigbee dongle hittas inte

Problem: ZHA hittar ingen dongle.

Lösning:

  • Koppla ur och in donglen
  • Starta om Raspberry Pi
  • Använd USB-förlängningskabel
  • Kontrollera att rätt port är vald
Sensor går inte att para

Problem: Sensorn dyker inte upp.

Lösning:

  • Håll in pairing-knappen längre
  • Flytta sensorn nära donglen
  • Byt batteri
  • Starta om ZHA
Sensor tappar kontakt

Problem: Sensorn försvinner ibland.

Lösning:

  • Lägg till Zigbee repeater (t.ex. smart plug)
  • Flytta donglen längre bort från Pi
  • Minska avståndet
Kamera funkar inte

Problem: Ingen bild visas.

Lösning:

  • Kontrollera att kameran är på nätverket
  • Testa ONVIF istället för RTSP
  • Kolla användarnamn/lösenord
  • Testa kamerans IP i webbläsare
Systemet är segt

Problem: Home Assistant känns långsamt.

Lösning:

  • Använd Ethernet istället för WiFi
  • Byt till SSD istället för SD-kort
  • Starta om systemet
Backup fungerar inte

Problem: Backuper misslyckas.

Lösning:

  • Kontrollera internetanslutning
  • Logga in igen i backup addon
  • Testa skapa backup manuellt
WiFi problem

Problem: Instabil uppkoppling.

Lösning:

  • Använd Ethernet istället
  • Flytta routern närmare
  • Undvik att köra både Zigbee och WiFi nära varandra
