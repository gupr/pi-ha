Raspberry Pi installation (Home Assistant OS)

Mål: Få igång Home Assistant på din Raspberry Pi.

  1. Ladda ner Raspberry Pi Imager på din dator
  2. Sätt i microSD-kortet i datorn
  3. Öppna Imager → välj:
    • Device: Raspberry Pi 5
    • OS: "Other specific purpose OS" → Home Assistant
    • Storage: ditt SD-kort
  4. Klicka på kugghjulet:
    • Sätt hostname (t.ex. "homeassistant")
    • Aktivera SSH (valfritt men bra)
    • Ställ in WiFi om du MÅSTE (annars använd kabel)
  5. Skriv OS till SD-kortet
  6. Sätt i SD-kortet i Pi
  7. Koppla till Pi:
    • Ethernet-kabel (rekommenderas starkt)
    • Strömadaptern
  8. Vänta 5–10 minuter
  9. Öppna i webbläsare:
    http://homeassistant.local:8123

Viktigt:

  • Använd helst inte WiFi för Pi → instabilt
  • Första boot tar lång tid → ha tålamod
  • SD-kort kan slitas → överväg SSD senare
⚙️ Home Assistant setup
  1. Skapa konto
  2. Ställ in:
    • Plats (viktigt för väder & tid)
    • Tidszon
    • Enheter
  3. Installera tillägg:
    • File Editor
    • Samba Share (för åtkomst från dator)
    • OBS! Zigbee finns redan integrerat i Home Assistant som ZHA
  4. Gå till Settings → System → Updates och uppdatera allt

Tips:

  • Använd mobilappen för enkel åtkomst
  • Ge systemet ett namn
🌐 Nätverk (superviktigt)
  • Pi bör vara ansluten via Ethernet
  • Kameror (speciellt PoE) kräver:
    • Router
    • PoE switch eller injector
  • Zigbee kan störas av WiFi:
    • Placera dongle bort från Pi
    • Använd USB-förlängning
💽 Lagring (viktigt för stabilitet)
  • SD-kort fungerar men slits över tid
  • Rekommenderat:
    • USB SSD
    • eller High Endurance SD-kort
  • Backup är extra viktigt om du kör SD
📡 Zigbee dongle (ZHA)
  1. Koppla in dongle i USB-förlängning
  2. Placera bort från Pi (minskar störningar)
  3. I Home Assistant:
    • Settings → Devices & Services
    • Add Integration
    • Välj ZHA
  4. Välj rätt seriell port (auto funkar oftast)

Viktigt:

  • Använd ZHA (inte deCONZ för detta projekt)
  • Se till att donglen inte sitter direkt i Pi
📶 Zigbee räckvidd (mesh)
  • Batterisensorer = dålig räckvidd
  • Behövs repeaters
  • Lösning:
    • Smart plug mellan hus och uthus
    • Bygger mesh automatiskt
  • Annars kan sensorer i uthus tappa kontakt
🌡️ Koppla sensorer
  1. Gå till ZHA → Add Device
  2. Sätt sensor i pairing mode (håll knapp)
  3. Vänta tills den dyker upp
  4. Ge tydligt namn:
    • Temp Inne
    • Temp Ute

Placering:

  • Ute:
    • Skydda från regn
    • Inte direkt sol
  • Inne:
    • Inte nära element
📷 IP-kamera
  1. Anslut kamera till nätverket
  2. Hitta IP-adress (router eller app)
  3. I Home Assistant:
    • Add Integration
    • Välj ONVIF (bäst) eller RTSP
  4. Ange:
    • IP
    • användarnamn/lösen

Viktigt:

  • Kameran måste stödja RTSP eller ONVIF
  • PoE är stabilare än WiFi
📊 Dashboard
  1. Gå till Dashboard → Edit
  2. Add Card
  3. Lägg till:
    • Temperature (inne)
    • Temperature (ute)
    • Luftfuktighet
    • Kamera feed
  4. Ordna snygg layout
📈 Grafer
  1. Add Card → History Graph
  2. Välj sensorer
  3. Ställ in tidsintervall
💾 Backup

Mål: Säkerställa att systemet kan återställas vid SD-krasch eller fel.

☁️ Moln-backup (rekommenderas)

  1. Gå till Settings → Add-ons
  2. Installera "Google Drive Backup"
  3. Logga in med Google-konto
  4. Aktivera schemalagd backup (t.ex. dagligen)
  • Fördel: säkert även om Pi går sönder
  • Nackdel: kräver internet

💽 Lokal backup (viktigt komplement)

  1. Gå till Settings → System → Backups
  2. Klicka "Create Backup"
  3. Ladda ner backup-filen till din dator

Alternativ:

  • Använd Samba Share addon:
    • Få tillgång till Pi via din dator
    • Kopiera backup-filer manuellt
  • Automatisera till NAS (om du har en):
    • Via SMB eller NFS

⚠️ Viktigt

  • Spara backup UTANFÖR Raspberry Pi (annars försvinner den vid krasch)
  • Testa att återställa minst en gång
  • Ha både lokal + moln backup
📱 Åtkomst från mobilen
  • Enklast: Home Assistant Cloud
  • Gratis alternativ:
    • Tailscale (rekommenderas)
    • Port forwarding (mer avancerat)
🤖 Automationer (valfritt)
  • Frostvarning
  • Hög luftfuktighet
  • Rörelselarm kamera
