Pi-Hem

Installation och drift av hemmaväderstation med Raspberry Pi

Inköpslista
Raspberry Pi installation (Home Assistant OS)

Mål: Få igång Home Assistant på din Raspberry Pi.

  1. Ladda ner Raspberry Pi Imager:
    Ladda ner Imager
  2. Sätt i microSD-kortet i datorn
  3. Öppna Imager och välj:
    • Device: Raspberry Pi 5
    • OS: Other specific purpose OS → Home Assistant
    • Storage: ditt SD-kort
  4. Klicka på kugghjulet (advanced settings):
    • Sätt hostname (t.ex. "homeassistant")
    • Aktivera SSH (valfritt men rekommenderas)
    • Ställ in WiFi endast om du inte kan använda kabel
  5. Skriv OS till SD-kortet
  6. Sätt in SD-kortet i Raspberry Pi
  7. Koppla in:
    • Ethernet-kabel (rekommenderas starkt)
    • Strömadapter
  8. Vänta 5–10 minuter på första uppstart
  9. Öppna i webbläsare:
    http://homeassistant.local:8123

Viktigt:

  • Använd Ethernet istället för WiFi för stabilitet
  • Första uppstarten tar tid – avbryt inte
  • SD-kort slits över tid – överväg SSD senare
Home Assistant setup
  1. Skapa konto
  2. Ställ in:
    • Plats (används för väder och tid)
    • Tidszon
  3. Installera tillägg:
    • File Editor
    • Samba Share (för åtkomst från dator)
  4. Gå till Settings → System → Updates och uppdatera allt

Tips:

  • Installera Home Assistant-appen på mobilen
  • Ge systemet ett tydligt namn
Nätverk
  • Raspberry Pi bör vara ansluten via Ethernet
  • Kameror kräver:
    • Router
    • PoE switch eller injector (om PoE används)
  • Zigbee kan störas av WiFi:
    • Använd USB-förlängning
    • Placera dongle bort från Pi
Lagring
  • SD-kort fungerar men har begränsad livslängd
  • Rekommenderat:
    • USB SSD (bäst)
    • eller High Endurance SD-kort
  • Backup är extra viktigt om du använder SD-kort
Zigbee dongle (ZHA)
  1. Koppla dongle via USB-förlängning
  2. Placera den bort från Raspberry Pi
  3. I Home Assistant:
    • Settings → Devices & Services
    • Add Integration → ZHA
  4. Välj seriell port (auto fungerar oftast)

Viktigt:

  • Använd ZHA (inte deCONZ i detta projekt)
  • Placering påverkar signalen mycket
Zigbee räckvidd (mesh)
  • Batteridrivna sensorer har begränsad räckvidd
  • Lösning:
    • Lägg till en Zigbee smart plug mellan hus och uthus
    • Den fungerar som repeater
  • Utan mesh kan sensorer tappa kontakt
Sensorer
  1. Gå till ZHA → Add Device
  2. Sätt sensorn i pairing mode (håll knapp)
  3. Vänta tills den dyker upp
  4. Döp sensorer tydligt:
    • Temp Inne
    • Temp Ute

Placering:

  • Utomhus:
    • Skydda från regn
    • Undvik direkt solljus
  • Inomhus:
    • Placera inte nära element
IP-kamera
  1. Anslut kameran till nätverket
  2. Ta reda på dess IP-adress
  3. I Home Assistant:
    • Add Integration
    • Välj ONVIF (rekommenderas) eller RTSP
  4. Ange:
    • IP-adress
    • Användarnamn och lösenord

Viktigt:

  • Kameran måste stödja RTSP eller ONVIF
  • PoE är stabilare än WiFi
Dashboard
  1. Gå till Dashboard → Edit
  2. Klicka Add Card
  3. Lägg till:
    • Temperatur inne
    • Temperatur ute
    • Luftfuktighet
    • Kamerabild
  4. Ordna layouten så den är lätt att läsa
Grafer
  1. Add Card → History Graph
  2. Välj sensorer
  3. Ställ in tidsintervall
Backup

Mål: Kunna återställa systemet vid fel eller krasch.

Moln-backup

  1. Installera Google Drive Backup addon
  2. Logga in
  3. Aktivera schemalagd backup

Lokal backup

  1. Settings → System → Backups
  2. Create Backup
  3. Ladda ner till dator

Viktigt

  • Spara backup utanför Raspberry Pi
  • Ha både lokal och moln-backup
  • Testa att återställa minst en gång
Åtkomst från mobilen
  • Enklast: Home Assistant Cloud
  • Gratis alternativ:
    • Tailscale (rekommenderas)
    • Port forwarding (mer avancerat)
Automationer (valfritt)
  • Frostvarning
  • Hög luftfuktighet
  • Rörelselarm via kamera
