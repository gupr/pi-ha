Raspberry Pi installation (Home Assistant OS)

Mål: Installera och starta Home Assistant på din Raspberry Pi.

  1. Ladda ner Raspberry Pi Imager:

    Ladda ner Imager
  2. Sätt i microSD-kortet i datorn
  3. Öppna Imager och välj:
    • Device: Raspberry Pi 5
    • OS: Other specific purpose OS → Home Assistant
    • Storage: ditt SD-kort
  4. Klicka på kugghjulet (Advanced settings):
    • Sätt hostname (t.ex. "homeassistant")
    • Aktivera SSH (rekommenderas)
    • Ställ in WiFi endast om kabel inte används
  5. Skriv OS till SD-kortet
  6. Sätt in SD-kortet i Raspberry Pi
  7. Koppla in:
    • Ethernet (rekommenderas)
    • Ström
  8. Vänta 5–10 minuter (första start tar tid)
  9. Öppna:
    http://homeassistant.local:8123

Viktigt:

  • Använd Ethernet för stabilitet
  • Avbryt inte första uppstarten
  • SD-kort slits → överväg SSD senare
Home Assistant setup
  1. Skapa konto
  2. Ställ in:
    • Plats
    • Tidszon
  3. Installera tillägg:
    • File Editor
    • Samba Share
  4. Settings → System → Updates → uppdatera allt

Tips:

  • Installera mobilappen
  • Döp systemet tydligt
Nätverk
  • Använd Ethernet till Raspberry Pi
  • Kameror kräver:
    • Router
    • PoE switch/injector (om PoE)
  • Zigbee kan störas av WiFi:
    • Använd USB-förlängning
    • Placera dongle bort från Pi
Lagring
  • SD-kort fungerar men slits
  • Rekommenderas:
    • USB SSD (bäst)
    • eller High Endurance SD
  • Backup är extra viktigt med SD
Zigbee dongle (ZHA)
  1. Använd USB-förlängning
  2. Placera bort från Pi
  3. I Home Assistant:
    • Settings → Devices & Services
    • Add Integration → ZHA
  4. Välj port (auto funkar oftast)

Viktigt:

  • Använd ZHA
  • Placering påverkar signal
Zigbee räckvidd (mesh)
  • Batterisensorer har kort räckvidd
  • Lösning:
    • Lägg till smart plug mellan enheter
    • Fungerar som repeater
Sensorer
  1. ZHA → Add Device
  2. Håll knapp (pairing)
  3. Vänta
  4. Döp tydligt:
    • Temp Inne
    • Temp Ute

Placering:

  • Utomhus: skydda från regn & sol
  • Inomhus: inte nära element
IP-kamera
  1. Anslut till nätverk
  2. Hitta IP-adress
  3. Add Integration:
    • ONVIF (bäst)
    • eller RTSP
  4. Fyll i login

Viktigt:

  • Måste stödja RTSP/ONVIF
  • PoE > WiFi
Dashboard
  1. Dashboard → Edit
  2. Add Card
  3. Lägg till:
    • Temperatur
    • Luftfuktighet
    • Kamera
Grafer
  1. Add Card → History Graph
  2. Välj sensorer
  3. Ställ tid
Backup

Mål: Kunna återställa systemet.

Moln

  1. Installera Google Drive Backup
  2. Logga in
  3. Aktivera schema

Lokal

  1. Settings → System → Backups
  2. Create Backup
  3. Ladda ner

Viktigt:

  • Spara utanför Pi
  • Ha minst 2 kopior
Åtkomst från mobilen
  • Enklast: Home Assistant Cloud
  • Gratis: Tailscale
  • Avancerat: Port forwarding
Automationer (valfritt)
  • Frostvarning
  • Hög luftfuktighet
