Hemmaväderstation

Inköpslista
1. Raspberry Pi installation
  • Installera Home Assistant via Raspberry Pi Imager
  • Sätt i SD-kort
  • Koppla nätverkskabel
  • Koppla ström
  • Öppna: http://homeassistant.local:8123
2. Home Assistant setup
  • Skapa konto
  • Ställ in plats
  • Kontrollera dashboard
  • Installera File Editor + Samba
3. Zigbee dongle
  • Koppla via USB-förlängning
  • Settings → Devices & Services
  • Add Integration → ZHA
4. Sensorer
  • Klicka “Add device”
  • Håll in knapp på sensor
  • Placera inne/ute korrekt
5. IP-kamera
  • Add Integration
  • ONVIF eller RTSP
  • Verifiera bild
6. Dashboard
  • Edit Dashboard
  • Lägg till temperatur inne/ute
  • Lägg till luftfuktighet
  • Lägg till kamera
7. Backup
  • Installera backup addon
  • Aktivera automatiska backuper
8. Grafer
  • Add Card → History Graph
  • Välj temperatur + fukt
